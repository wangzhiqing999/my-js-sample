离线转换处理 



----------
前置准备工作
----------
1.安装 node.js


2.配置淘宝定制的 cnpm (网络卡的原因)
npm install -g cnpm --registry=https://registry.npm.taobao.org
npm config set registry https://registry.npm.taobao.org


3.安装Babel命令行工具
cnpm install -g babel-cli



----------
使用 babel ， 将 babel 转换为标准的 JavaScript
----------

方案一. 使用 babel-plugin-transform-react-jsx 插件.
cnpm install babel-plugin-transform-react-jsx

babel --plugins transform-react-jsx  S0002_HelloWorldReact.js  --out-file S0003_HelloWorld.js



方案二. 使用 babel-preset-react 插件.
cnpm install babel-preset-react

babel S0002_HelloWorldReact.js --presets react --out-file S0003_HelloWorld.js



----------
修改 html 页面， 使用处理后的 js 文件. 并移除 type="text/babel"
